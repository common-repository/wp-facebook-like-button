== Description ==

The Like button enables users to make connections to your pages and share content back to their friends on Facebook with one click.

Just activate the plugin and visit settings panel to configure the output.

More info at: http://www.dolcebita.com/wordpress/facebook-like-button/

== Installation ==

1. Upload `wp-facebook-like-button` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use settings panel to configure the output or call the function inside your theme.
